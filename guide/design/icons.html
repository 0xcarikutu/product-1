Icon Library

The Datto Icon Library is used as a central repository for product icons that help with direction and instruction of how to use our products. Our goal is to use these icons to help with bringing cohesive usability and behaviour to all of our product lines. We have setup Fontastic to prefix all icons with the .icon-* class name for consistency.
