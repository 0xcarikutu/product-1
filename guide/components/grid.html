Grid

We use grids exclusively for layouts which require a number of columns of fluid width. Grids can have 1-12 columns using the Boostrap 3 grid system. The Bootstrap grid accounts for all screen sizes by using .col-xs- (phones), .col-sm- (tablets), .col-md- (desktop) and .col-lg- (large screens). It is important with this grid system your column count always is equal to 12.
