The following steps install software (an agent) on your target system that enables backups to your Datto appliance and the Datto cloud.

(icon) Before proceeding, please review the Predeployment Checklist.

How do we communicate with the system?

To backup your system, Datto can either install an agent on the system or, for customers using VMWare, use the hypervisor to back up you system. Agents should be installed on your system before going through this wizard, and can be downloaded from partners.datto.com/downloads.
